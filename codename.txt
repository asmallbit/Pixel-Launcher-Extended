#NexusLauncherRelease Codes:
NexusLauncherRelease(xy) format is used. x & y are variables, where each different values of these variables define something.

• If x=0, then launcher doesn't have any material you greetings.
• If x=1, then launcher contains material you greetings style 1.
• If x=2, then launcher contains material you greetings style 2.

• If y=0, then launcher is for Android 13.
• If y=1, then launcher is for Android 13 QPR.
• If y=2, then launcher is for Android 13 QPR2.

• Example Of Style 1
Line 1 - Material You Greetings
Line 2 - Day & Date
Line 3 - Weather Information

• Example Of Style 2
Line 1 - Material You Greetings, Day & Date
Line 2 - Weather Information
